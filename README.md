# EcoTrack AI

EcoTrack AI is a web application that helps users understand, track, and reduce their carbon footprint through personalized insights and sustainability recommendations.

## Features

* Carbon footprint calculator
* Personalized sustainability recommendations
* Emission tracking dashboard
* Data visualization with charts
* User-friendly interface

## Technology Stack

* Python (Flask)
* HTML, CSS, JavaScript
* SQLite Database
* Gemini AI API
* Chart.js

## Project Structure

* `app.py` – Main Flask application
* `calculator.py` – Carbon footprint calculation logic
* `db.py` – Database operations
* `gemini_service.py` – AI recommendation service
* `templates/` – HTML templates
* `static/` – CSS and JavaScript files
* `tests/` – Unit tests

## Installation

```bash
pip install -r requirements.txt
python app.py
```
